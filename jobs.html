As a Junior/Graduate Front End Web Developer at CTL Australia, you will collaborate closely with our experienced team to develop and maintain captivating web applications using React.js. Key Responsibility

  1. Collaborate with cross-functional teams to design, develop, and implement user-friendly web interfaces using React.js and React Native.
  2. Troubleshoot full-stack issues, with a focus on Node.js RESTful APIs and cloud services such as AWS or Azure for optimizing application performance and scalability.

Preferable Attributes

  • Experience with responsive design and mobile-first development.
  • Familiarity with modern web development tools and libraries, understanding of UI/UX principles, and knowledge of Agile methodologies.
